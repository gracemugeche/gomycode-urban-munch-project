PricingCard Component Demo

Interactive examples of the reusable pricing card component

1. Basic Usage

Simple example showing how to create pricing cards with the reusable component.

// Create a pricing card const card = new PricingCard({ title: "Basic Plan", price: 9.99, currency: "$", period: "/month", features: ["1 GB Storage", "Basic Support", "All Core Features"], buttonText: "Start Trial", onButtonClick: (data) => console.log('Selected:', data) }); // Add to container document.getElementById('container').appendChild(card.render());

2. Featured Card with Multiple Plans

Multiple pricing cards with one marked as featured. The featured card has a special highlight.

// Using PricingCardContainer for multiple cards const container = new PricingCardContainer(document.getElementById('container')); // Add multiple cards container.addCard({ title: "Basic", price: 9.99, features: ["1 GB Storage", "Basic Support"] }); container.addCard({ title: "Pro", price: 19.99, featured: true, // This will be highlighted features: ["10 GB Storage", "Priority Support", "Advanced Analytics"] });

3. Dynamic Updates

Demonstrating how to update pricing cards dynamically using the built-in methods.

// Update methods available: card.updatePrice(15.99); card.updateTitle("Updated Plan"); card.setFeatured(true, "Best Value"); card.updateFeatures(["New Feature 1", "New Feature 2"]);

4. Custom Styling & Events

Cards with custom CSS classes and custom click handlers.

// Custom styling with CSS classes const card = new PricingCard({ title: "Premium", price: 29.99, className: "dark-theme", // Add custom CSS class onButtonClick: (data) => { // Custom click handler showCustomModal(data); } });
Usage Tips:
  • Include the PricingCard.js file in your project
  • Use new PricingCard(options) for single cards
  • Use PricingCardContainer for managing multiple cards
  • All update methods work dynamically after rendering
  • Custom CSS classes can be added via the className option
  • Click handlers receive the full card data object
